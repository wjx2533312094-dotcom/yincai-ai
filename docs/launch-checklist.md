# 影策 AI 正式上线清单

## 品牌和版权

- 确认产品名、Logo、域名未侵犯他人商标或已有品牌。
- 首页图片、图标、字体、文案均使用自有或可商用授权素材。
- 保存 AI 图片生成记录和最终素材来源记录。
- 页面中加入“AI 辅助生成，使用前请人工审核”的提示。

## 账号和隐私

- 将本地 `.data/auth.json` 替换为正式数据库。
- 手机验证码接入真实短信服务商，不再向前端返回 `devCode`。
- 验证码需要频率限制、过期时间、错误次数限制。
- 配置用户协议、隐私政策和 AI 内容说明入口。
- 明确手机号、订单、生成记录的保存期限和删除方式。

## 支付和套餐

- 将 `/api/billing/activate` 的模拟开通替换为真实支付回调。
- 接入微信支付、支付宝或 Stripe 后，必须以支付平台服务端回调为准发放余额。
- 保存订单号、支付状态、套餐权益、退款状态和发票信息。
- 购买页展示价格、权益、有效期、退款规则和客服入口。

## AI 服务

- 使用服务端环境变量保存 `OPENAI_API_KEY`、`OPENAI_API_BASE_URL`、`OPENAI_MODEL`。
- 不在前端、日志、报错中输出完整 API Key。
- 增加请求超时、错误提示、重试和模型服务不可用时的提示。
- 增加内容安全策略，拦截明显违法、侵权、仿冒和高风险生成请求。

## 部署和运维

- 生产环境使用 HTTPS。
- 中国大陆服务器通常需要 ICP 备案。
- 配置日志、错误监控、数据库备份和密钥轮换。
- 配置环境变量分组：开发、测试、生产。
- 上线前用真实支付沙箱、真实短信测试号和小额套餐完整跑通。
